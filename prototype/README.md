# Prototype 1: Low Connectivity Banner & Feedback

Interactive simulation of how Sitetracker Mobile should communicate slow/offline network states to field users during normal app usage.

See the [root README](../README.md) for full documentation, setup instructions, and how to use.

## Quick Start

```bash
npm install
npm run dev
# Open http://localhost:5173
```

## What to try

1. Use the **Simulate Network** panel (top-left) to switch between Good / Slow / Offline
2. Tap the yellow banner to open the connectivity detail sheet
3. Navigate between Home and Job Detail screens to see the banner in context
